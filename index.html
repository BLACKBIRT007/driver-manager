What it does

Simple driver management with clean release handling

🔍

Full device scan

Detects hardware through Windows tools and lists devices, driver versions, manufacturers, and IDs.

📡

Driver checking

Checks for newer driver versions and shows what can be updated before you install anything.

⬇

Latest download link

The download button always points to the newest GitHub Release asset with a fixed filename.

🗂

Legacy versions

Older builds stay available automatically on the GitHub Releases page. No messy EXE folders in the repo.

⚙

Manual build button

The build button opens the GitHub Actions workflow where the owner can run a new build safely.

🔄

Soft self-update

The launcher can check GitHub Releases on startup, download a newer installer, install silently, and restart.

How it works

The release system

1

You update the code

Edit the Python files locally, test them, and push the changes to the main branch.

2

You create a version tag

Push a tag like v1.0.1, or manually run the GitHub Actions workflow from the build button.

3

GitHub builds the installer

The workflow builds DriverHandlerByCROD.exe, Launcher.exe, and the setup installer.

4

Latest becomes latest, old becomes legacy

The newest release becomes the download button target. Older releases remain in the legacy list.

5

Installed apps can soft-update

Launcher.exe checks GitHub Releases on startup and can install newer versions silently.

Installation

Install once, update through releases

1

Click Download latest to get the newest installer.
File name: Driver_Handler_By_CROD_Setup.exe

2

Run the installer and follow the setup wizard.
If Windows SmartScreen appears, click More info → Run anyway.

3

Keep Start with Windows enabled if you want background checks and soft app updates.

4

Open the app, scan devices, and test before enabling automatic driver installation.

Driver updates can break hardware if the wrong driver is installed. Keep automatic driver installation off until scanning, matching, restore points, and rollback are fully tested.
