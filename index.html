What it does

Everything in one place

🔍

Full Device Scan

Detects every piece of hardware on your PC — graphics cards, audio chips, Wi-Fi adapters, USB controllers and more — using Windows built-in tools.

📡

Official Driver Sources

Checks Microsoft's Update Catalog for newer drivers. The same source Windows Update uses — safe and official.

⬇

One-Click Updates

Update all outdated drivers at once, or pick exactly which ones you want. Progress shown in real time.

🤫

Silent Background Mode

Runs from Windows startup in the system tray. Checks for driver updates automatically and installs them without interrupting you.

🔄

Self-Updating

The app updates itself silently on startup. You always have the latest version without doing anything.

🆓

Completely Free

No subscription, no trial, no account needed. Open source on GitHub — you can see exactly what it does.

How it works

Simple from start to finish

1

Install once

Download and run the installer. It takes under a minute and sets everything up — including starting automatically with Windows.

2

Automatic scanning

Every time Windows starts, the app scans your hardware and checks for newer drivers in the background — no windows, no interruptions.

3

Get notified

A notification appears in your system tray when updates are found. Click to open the app and see exactly what's available.

4

Update your way

Click Update All to install everything, or pick specific drivers. Or turn on auto-install and never think about it again.

Installation

Up and running in 2 minutes

1

Click Download for Windows above to get the installer
DriverManager_Setup.exe — about 30 MB

2

Run the installer and follow the setup wizard
If Windows shows a SmartScreen warning, click "More info" → "Run anyway"

3

Leave "Start with Windows" checked for automatic background updates

4

Click Scan Devices to see your hardware and available updates
